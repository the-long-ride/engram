# Вкладка Runtime

Вкладка Runtime представляет собой доступный только для чтения отчет о разрешенной конфигурации и путях. Считайте ее первой страницей для устранения неполадок.

## Группы отчетов времени выполнения

Отчет группирует разрешенные значения для:

- **Profile** — активный профиль и источник разрешения
- **Memory roots** — пути к памяти рабочей области и глобальной памяти
- **Core config** — включено, область действия, чтение, подтверждение, роли
- **Routing** — лимит загрузки, параметры графа, векторные настройки
- **Graph** — включено, макс. связанных, мин. оценка
- **Git detection** — удаленный репозиторий, URL удаленного репозитория, ветка, автосинхронизация

Каждый вывод объясняет, что Engram фактически разрешил, а не только то, что было настроено. Используйте для отладки поведения профиля, корня, Git, маршрутизации и хуков.

## Закрыть сервер (Close server)

Останавливает локальный сервер Entry. Используйте для обеспечения безопасности после работы с настройками.

<RiskCallout level="risky">
Панель предназначена только для локального использования. Закройте сервер на вкладке Runtime по окончании работы во избежание утечек.
</RiskCallout>

## Эквивалент в CLI

```bash
engram config view
engram entry
```
