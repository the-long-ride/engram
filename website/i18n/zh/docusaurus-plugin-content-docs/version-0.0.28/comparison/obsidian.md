# Obsidian

Obsidian 是一个非常受欢迎的构建在本地 Markdown 文件之上的个人知识库。Engram 也是文件优先和基于 Markdown 的，但侧重点不同。

| 维度 | Engram | Obsidian |
| --- | --- | --- |
| 目的 | AI Agent 内存协议 | 个人知识库 |
| 路由 | 将图感知任务路由到 Agent 上下文中 | 手动链接和搜索 |
| 审批 | 在持久写入前的人类审批门 | 人类直接创作 |
| Agent 集成 | 技能文件、斜杠适配器、MCP、钩子 | 无内置集成 |
| 最合适 | 使用 AI Agent 且需要持久路由内存的团队 | 构建关联笔记图的个人 |

同时使用两者：将 Engram 用于面向 Agent 的持久内存，将 Obsidian 用于个人知识。两者都对 Markdown 和 Git 友好。
