# Tolaria

Tolaria는 에이전트 공간에서 메모리에 중점을 둔 또 다른 도구입니다. Engram은 Git 네이티브 감사 기록 및 에이전트와 무관한 어댑터와 함께 인간의 소유권 및 파일 우선 Markdown을 계약으로 삼는다는 점에서 다릅니다.

메모리가 저장소에서 읽을 수 있는 프로토콜이 되기를 원하는 경우 Engram을 사용하십시오: Markdown 우선, 인간 승인, Git 검토, 실행 중인 서버 없이도 에이전트 간 이식 가능.
