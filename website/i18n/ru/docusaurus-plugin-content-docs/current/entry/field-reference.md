# Полный справочник полей

Эта страница является каноническим справочником полей конечного пользователя для каждого поля ввода и элемента управления Entry Web UI.

## Как читать этот справочник

Каждое поле содержит:

- **Ключ конфигурации** — ключ, используемый в конфигурационных файлах и CLI
- **Элемент управления** — тип ввода
- **По умолчанию** — безопасное значение по умолчанию
- **Риск** — `normal`, `caution` (предупреждение) или `risky` (рискованно)
- **Примечания** — что делает поле и когда его менять

## Core (Ядро)

| Ключ конфигурации | Элемент управления | По умолчанию | Риск | Примечания |
| --- | --- | --- | --- | --- |
| `enabled` | переключатель | `true` | risky | Главный выключатель. Отключение останавливает работу Engram. |
| `scope` | выбор | `both` | risky | Направление сохранения: `workspace`, `global`, `both`. |
| `read` | выбор | `auto` | normal | Когда хуки внедряют память: `auto`, `startup`, `always`, `manual`, `off`. |
| `proof` | выбор | `off` | normal | Строка подтверждения хука: `off`, `compact`. |
| `global_path` | текст | пусто | risky | Путь к файловой системе для глобальной памяти. |
| `default_profile` | выбор | пусто | risky | Профиль, используемый, когда ни один не задан явно. |
| `roles` | роли | пусто | normal | Имена ролей через запятую для маршрутизации. |
| `theme` | выбор | `dark` | hidden | Внутреннее/скрытое. Не для пользователя. |
