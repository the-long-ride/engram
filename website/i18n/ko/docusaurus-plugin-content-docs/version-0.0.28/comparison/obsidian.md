# Obsidian

Obsidian은 로컬 Markdown 파일을 기반으로 구축된 널리 쓰이는 개인 지식 베이스입니다. Engram 역시 파일 우선 및 Markdown 기반이지만 초점이 다릅니다.

| 차원 | Engram | Obsidian |
| --- | --- | --- |
| 목적 | AI 에이전트 메모리 프로토콜 | 개인 지식 베이스 |
| 라우팅 | 그래프 인식 작업 라우팅을 에이전트 컨텍스트에 주입 | 수동 링크 및 검색 |
| 승인 | 영구 쓰기 전 인간 승인 게이트 | 인간이 직접 작성 |
| 에이전트 통합 | 기술 파일, 슬래시 어댑터, MCP, hook | 내장 기능 없음 |
| 가장 적합한 대상 | 라우팅된 영구 메모리가 필요한 AI 에이전트를 사용하는 팀 | 연결된 메모 그래프를 구축하는 개인 |

두 가지 모두 사용하십시오: 에이전트용 영구 메모리에는 Engram을, 개인 지식에는 Obsidian을 사용합니다. 둘 다 Markdown 및 Git 친화적입니다.
