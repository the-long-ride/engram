# Tolaria

Tolaria は、エージェント空間におけるもう 1 つのメモリ重視のツールです。Engram は、Git ネイティブの監査履歴とエージェントに依存しないアダプターを備え、人間の所有権とファイル優先の Markdown を契約とすることで差別化しています。

メモリをリポジトリから読み取り可能なプロトコルにしたい場合は、Engram を使用します。Markdown ファースト、人間の承認、Git によるレビュー、実行中のサーバーがなくてもエージェント間で移植可能です。
