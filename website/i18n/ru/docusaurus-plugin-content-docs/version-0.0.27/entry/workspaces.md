# Вкладка Workspaces

Вкладка Workspaces регистрирует репозитории проектов и управляет их состоянием связи.

## Имя рабочей области (Workspace name)

Удобное отображаемое имя для пути к репозиторию/проекту. Держите его кратким и легко узнаваемым.

## Путь к рабочей области (Workspace path)

Путь файловой системы к репозиторию/проекту. Убедитесь, что папка существует или может быть инициализирована; избегайте системных папок.

## Привязать / Отвязать (Link / Unlink)

Активно ли Engram подключает сгенерированные инструкции и хуки к рабочей области. Привязывайте активные репозитории; отвязывайте заархивированные или тестовые репозитории.

<RiskCallout level="caution">
Отвязка останавливает получение агентами инструкций Engram. Подтвердите перед отвязкой активной рабочей области.
</RiskCallout>

## Удалить (Delete)

Удаляет регистрацию рабочей области. Уточните, удаляет ли это только регистрацию или файлы памяти; документация должна соответствовать реализации. Предпочитайте отвязку удалению для возможности последующего аудита.

## Эквивалент в CLI

```bash
engram inject
engram link codex
engram unlink
```
